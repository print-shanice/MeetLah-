## ğŸ”¥ Monthly Streak System

### What is it?
The Monthly Streak tracks how many consecutive months your group has met up. Meeting at least once per month keeps the streak alive!

### How it works
1. **Automatic Tracking:** When you create a meetup event, the system automatically checks if it's a new month
2. **Streak Increment:** If you met last month and meet this month, your streak increases by 1
3. **Streak Reset:** If you skip a month, the streak resets to 1 on your next meetup
4. **Record Keeping:** The system tracks both your current streak and your longest streak ever

### Where to see it
- **Sidebar:** The streak display appears at the top of the sidebar
- **Stats Shown:**
  - Current streak (with fire emoji ğŸ”¥ when active)
  - Longest streak record
  - Last meetup month
  - Achievement badges

### Achievements
- ğŸ¥‰ **3-Month Warrior:** Keep a 3+ month streak
- ğŸ¥ˆ **Half-Year Hero:** Reach 6 consecutive months
- ğŸ¥‡ **Year-Long Legend:** Achieve 12 months in a row
- ğŸ’ **2-Year Champion:** Maintain 24+ months
- ğŸ“ˆ **Record Holder:** When current streak = longest streak

### Tips
- Create at least one meetup event each month
- The streak updates instantly when you create a meetup
- Personal events don't count - only meetup events affect the streak
- All calendar members benefit from the same streak

## ğŸ“… Calendar Sharing

### Creating a Shared Calendar
1. Click "Create New Calendar" from the dropdown menu
2. Give it a meaningful name (e.g., "College Friends", "Family")
3. You're automatically added as the owner and first member

### Sharing Your Calendar
1. Click **"Share Calendar"** in the sidebar
2. A modal appears with a unique share code (e.g., "a3f8k2m9p1")
3. Share this code with friends via:
   - Text message
   - WhatsApp
   - Email
   - Social media

### Joining a Calendar
1. Click the calendar dropdown menu
2. Select **"Join Calendar"**
3. Enter the share code you received
4. Click "Join Calendar"
5. You're now a member and can see all events!

### Member Management
- **Owner:** Creator of the calendar (can't be changed)
- **Members:** Anyone who joins via share code
- **Leaving:** Members can leave a calendar anytime
- **Colors:** Each member gets a random color for their events
- **You can change your own color** in the sidebar

## ğŸ¤ Meetup Creation

### What is a Meetup?
A meetup is a special event where ALL calendar members are invited. It's different from personal events which only affect one person.

### Creating a Meetup

#### Method 1: Direct Creation
1. Click **"Create Meetup"** in the sidebar
2. Select a date and time
3. Fill in:
   - **Title:** Name of the meetup (e.g., "Movie Night")
   - **Location:** Where you're meeting (e.g., "Cinema at Mall")
   - **Start Time:** When to meet
   - **End Time:** Expected end time
4. Click "Create Meetup"

#### Method 2: From Availability View
1. Click **"Check Availability"** in sidebar
2. Find a green time slot (when everyone is free)
3. Click on the slot
4. Fill in meetup details
5. Create!

### Meetup Features
- **Purple Color:** Meetups appear in purple (not member colors)
- **All Members Included:** Everyone in the calendar is automatically added
- **Clash Detection:** Can't create if someone has a conflicting event
- **Streak Tracking:** Creating a meetup updates the monthly streak
- **Attendance Tracking:** After the meetup, track who was late

### Best Practices
- Check availability before creating a meetup
- Add specific location details
- Give enough advance notice
- Update/delete if plans change

## ğŸ‘¥ Availability Checking

### What is it?

### How to Use
1. Click **"Check Availability"** in the sidebar
2. The calendar switches to Availability View
3. You'll see:
   - **Green areas:** Everyone is free âœ…
   - **Red areas:** Someone is busy âŒ
   - **Yellow areas:** Partially available

### Creating from Availability
1. Click on any green (available) slot
2. The Create Meetup modal opens
3. The selected time is pre-filled
4. Complete the details and create

### Tips
- Longer meetings need longer green blocks
- Consider time zones if members are in different locations
- Weekend availability is often better
- Check a few weeks ahead for better options

### Toggle Back
Click the close button or header to return to normal calendar view
